### 如何创建N阶数组

- 使用双层循环
- 或者使用fill+map

```javascript
function createNN(n){
    return new Array(n).fill(null).map(
        ()=>{
        return new Array(n).fill(0)
        }
    );
}
```

- 这里需要注意一个问题，当return语句一行没写完时，一定要在结束时加上分号，这是因为JS引擎会自动在return哪一行的后面加上一个分号
