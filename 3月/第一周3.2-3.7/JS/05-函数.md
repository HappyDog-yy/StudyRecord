### 函数定义的三种方式

- 1.箭头函数(参数列表)=>{函数体}
- 箭头函数是匿名函数，并且没有this指向

- 2.函数声明
- 任何函数都是通过new一个Function()来实现的
- 为什么我们平时的函数声明不需要new一个function函数
- 因为平时的函数声明是new Function()的语法糖

- 3.函数表达式
- 函数表达式是将函数赋给一个变量
- 用变量来存储这个函数

### return与返回值之间添加新行的后果

！！！注意不要在return与返回值之间添加新行。
- 因为JavaScript默认会在return之后添加分号
- 这样一来，你的返回值变成undefinted
- 同时return之后的语句再不也不会执行

### 函数名最好见名思意

- showMessage():展示信息
- getAge():得到年龄age
- calcSum():计算总和
- createForm():创建表单
- checkPermission():检查权限

### 立即执行函数

```javascript
(function (a,b){
    return a+b;
})();
```

- 可以创建一个单独的隔离的作用域
- 出现主要是为了解决es5时代变量无块级作用域污染全局的问题
- 现在可以使用let定义变量解决该问题
- 注意最后的分号不能少
